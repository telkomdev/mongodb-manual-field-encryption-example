## MongoDb Manual Field Level Encryption

## Encryption Library/Package

- Golang https://github.com/telkomdev/go-crypsi
- NodeJs https://github.com/telkomdev/crypsi

## Available Examples

- Golang
- NodeJs

## Getting Started

Start Mongo Docker

```shell
$ docker-compose up
```

Run `Golang` Example
```shell
$ go run main.go
```

Run `NodeJs` Example
```shell
$ npm start
```
